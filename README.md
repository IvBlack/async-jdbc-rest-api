# Async REST API in Java.
Application's effectively deleting multiple rows 
from a table in async mode.

+++++++++++++++++++++++++++++++++++++++++
- Asynchronous
- Native SQL queries
- Datasource
- Connection Pool
- Flyway and JDBC Template
+++++++++++++++++++++++++++++++++++++++++

# How to get started
- connect to postgresSQL with creds into application.yaml
- mvn clean install
- run app
